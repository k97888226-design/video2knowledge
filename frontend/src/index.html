API 接口文档

GET /api/v1/health

系统健康检查

curl http://localhost:8000/api/v1/health

POST /api/v1/video/info

获取视频信息

curl -X POST "http://localhost:8000/api/v1/video/info?url=VIDEO_URL"

POST /api/v1/upload/process

上传字幕，或上传视频并附带字幕（异步任务）

curl -X POST http://localhost:8000/api/v1/upload/process \
  -F "file=@./sample.srt" \
  -F "language=zh" \
  -F "export_formats=markdown,json"

GET /api/v1/task/{task_id}

查询任务状态

curl http://localhost:8000/api/v1/task/TASK_ID
