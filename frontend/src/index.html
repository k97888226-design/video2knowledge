API 接口文档

GET /api/v1/health

系统健康检查

curl http://localhost:8000/api/v1/health

POST /api/v1/video/info

获取视频信息

curl -X POST "http://localhost:8000/api/v1/video/info?url=VIDEO_URL"

POST /api/v1/video/process

处理视频（异步任务）

curl -X POST http://localhost:8000/api/v1/video/process \
  -H "Content-Type: application/json" \
  -d '{"url": "VIDEO_URL", "language": "auto", "export_formats": ["markdown", "json"]}'

GET /api/v1/task/{task_id}

查询任务状态

curl http://localhost:8000/api/v1/task/TASK_ID

POST /api/v1/batch/process

批量处理视频

curl -X POST http://localhost:8000/api/v1/batch/process \
  -H "Content-Type: application/json" \
  -d '{"urls": ["URL1", "URL2"], "language": "auto"}'
